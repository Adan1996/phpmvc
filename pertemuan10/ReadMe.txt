Flash Message
Pesan kilat yang digunakan ketika user telah mengerjakan salah satu aksi di dalam CRUD baik itu tambah data,
hapus data, dan ubah data. Dalam materi ini developer akan menggunakan konsep "$_SESSION"

Tutorial :
- Buat file Flasher.php pada folder core
- Buat Class Flasher dan method setFlash() & flash()
- Tambah static method Flasher::flash() pada view mahasiswa index
- Tambah static method Flasher::setFlash() pada controller Mahasiswa
